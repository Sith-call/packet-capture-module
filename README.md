# packet-capture-module

## 개요
http protocol과 dns protocol의 패킷을 캡처하여 분석

## 폴더 설명
* header : 참고한 http protocol의 헤더
* image : 실제 데모 및 패킷 캡처 이미지
* pcap : 캡처한 패킷들의 pcap 파일
* reference : 참고한 코드
